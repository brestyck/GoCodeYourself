# Суффиксное дерево

## Бор

Бор (trie) - структура для хранения строк, дерево с символами на ребрах.
Проще говоря, дерево для строк.

Возьмем слова:
`ababa`
`bbaa`
`baaba`
`abba`
`aa`

Строим граф для **ababa** и **bbaa**

![Бор](../imgs/bohr_1.png)

В этом дереве:
Ребра - элементы строк (_буквы_) и переход от одного состояния к другому
Вершины
