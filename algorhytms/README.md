# Алгоритм Рабина-Карпа

1. Вычисляем хэш
```
pattern text[len(pattern)]
```
2. Циклом сравниваем хэш, и, если совпадает, проверяем посимвольно во избежание _коллизии_.

Мы берем **m** - длину паттерна, чтобы не высчитывать хэш всего текста.
### Пример
Найти фрагмент
> abcd
В строке типа
> bcabcdabcccbdabc
>
